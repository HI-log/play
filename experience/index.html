오늘 어디 갈까요?

현재 위치 + 실시간 날씨를 보고 가까운 곳 3개 · 먼 곳 2개를 추천해요.
위치 권한이 필요해요. 거부해도 도시를 직접 고를 수 있습니다.

또는 도시 선택:

2~4세 데려갈 때 체크리스트

준비물

  • 여벌옷·물티슈 — 동물 먹이주기·갯벌·수확체험은 옷이 무조건 더러워짐
  • 장화/슬립온 — 갯벌·진흙 농장은 운동화 비추
  • 모자·자외선차단제 — 야외 목장은 그늘 부족
  • 유모차 — 양떼목장 산책로는 경사 있음 → 힙시트 병행 권장

시즌

  • 4~6월 / 9~10월 — 야외 목장 골든시즌
  • 11월~5월 — 딸기 수확체험
  • 봄~가을 — 갯벌체험 (썰물 시간 확인 필수)
  • 겨울·우천 — 실내 동물원·박물관·과학관으로 대체

예약 팁

  • 국립박물관/과학관 어린이관 — 사전예약 필수, 회차제 운영
  • 천문대 — 야간 운영(금·토 위주), 사전예약 필수
  • 키자니아 — 평일·오전권이 저렴, 만 3세부터 가능
  • 요리/공예 클래스 — 만 2~3세 가능 클래스인지 연령 제한 확인

주의

  • 대형 가축(소·말) 가까이 갈 땐 보호자 동반 필수
  • 실내 동물 카페는 위생 정책·연령 제한 확인 (일부는 36개월 이상)
  • 갯벌은 썰물 타이밍 못 맞추면 진흙뿐 → 물때표 확인
  • 천문대 별 보기는 날씨 변수 큼 → 환불 정책 확인
